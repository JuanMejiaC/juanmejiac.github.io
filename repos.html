PacMen Exercise

In this project I am building a Pacman game. Once running it will display 2 buttons. Everytime the Add Pacman button is clicked it will create a new pacman. When the button Start Game is clicked it will start the game. The pacmen bounce on contact with the browser edges. The regular pacmen will change to Thumb Up pacman randomly, when that happens the player will have a moment to capture the pacman that changed by triggering mousedown right on it before it changes back to the regular pacman. When a pacman is capture all the pacmen will stand still until the mouse button is released. If a regular pacman is clicked on the game epeed will increase, so the pacmen will move faster. Eyes Exercise

In this project named Eyes, I am building a human face that can perform all the facial expressions that humans do. When click on an eye it will wink. When click on an eyebrow it will perform an eyebrow flash (raise and lower quickly). The eyes will also follow the cursor movements. Eye-Exercise

Real Time Bus Tracker

In this project I am building a real time Bus Tracker in the area of Boston. Currently it only shows the stops and buses of route 1 serving the route Harvard Square-Nubian Station. You can click on a stop icon and it will show the stop id, street where it is located and whether the location has wheelchair boarding or not. When a bus icon is clicked it will show the route id, the current status witch respect to the nearest stop and the occupancy status. Real-Time-Bus-Tracker
